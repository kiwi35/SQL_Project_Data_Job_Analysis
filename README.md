## Customizing the queries
These queries filter on specific columns you can change:

- **Role** (`job_title_short`): one of `Data Analyst`, `Data Engineer`, 
  `Data Scientist`, `Senior Data Analyst`, `Senior Data Engineer`, 
  `Senior Data Scientist`, `Machine Learning Engineer`, `Cloud Engineer`, 
  `Software Engineer`, `Business Analyst`
- **Location** (`job_location`): `Anywhere` = remote; otherwise any city/state/country present in the data
